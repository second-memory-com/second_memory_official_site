# Second Memory 公式サイト 設計図

「Second Memory」公式サイトの立ち上げに向けた、初期段階の設計図です。

## プロジェクトの目的
「Second Memory」というプロダクトまたはブランドの公式ウェブサイトを構築し、その価値や情報を効果的に伝えます。

## 決定事項（要件定義: コーポレート・自然調和路線へ修正）
以下の内容で「Second Memory株式会社」公式サイトの構築を進めます。
- **コンセプト**: 「身体の記憶（Second Memory）から、脳を調律する」認知メカニズムからヒントを得たコンディショニング・システム（ソフトウェア・映像コンテンツ）
- **ターゲット（コア層）**: 発達障害のグレーゾーンに位置し、高い論理的思考力や創造力を持ちながらも、不注意や人間関係の摩擦により社会適応に苦感している方。（特に初期アプローチとして**「ひきこもり」**状態にある方を想定）
- **表現方針**: 医療的な治療・矯正ではなく、「本来のポテンシャルを取り戻し、運転感覚を自分の手に取り戻す」あるいは「モチベーションの源泉を掘り起こす」というポジティブな表現へ昇華する。
- **独自性と強み**: 
  - 競合の多い「ADHD専門」のアプローチとは一線を画し、「室内でシステムに向かって黙々と行える身体的アプローチ」である点。
  - 継続する（2〜4ヶ月）ことで、思考のノイズが取れるだけでなく「意欲（モチベーション）」が自然と生まれる点。
- **今後の展望（B2B/B2G）**: 自宅（室内）で完結できるメリットを活かし、将来的に各市町村の「ひきこもり支援センター」など行政機関や団体への導入を目指す。
- **デザイン**: 「自然の中での深呼吸」「霧が晴れてフォーカスが合う感覚」を表現する、ナチュラルで透明感のあるデザイン。室内で行う静かで研ぎ澄まされたコンディショニングを演出。
- **主要コンテンツ**: コーポレートサイトとしての信頼感を持たせる構成。個人ユーザーへの訴求と、将来の支援団体（B2B/B2G）へ向けた信頼性の担保を両立させる。

## 提案する構成（暫定）

### 技術スタック
モダンで高品質なデザインを実現するため、以下の構成を提案します。
- **フロントエンド**: HTML5, Vanilla CSS（※初心者が理解しやすいようJSや複雑な装飾は極力排除したシンプル設計）
- **デザイン**: プレミアムで動きのある、レスポンシブなデザイン
- **ホスティング（将来案）**: Netlify, Vercel 等

### セキュリティ・ガイドライン（初心者向け安全設計）
初めてサイトをネットに公開する際でも、安全に運用するための絶対のルールです。

1. **APIキーやパスワードの直書き禁止（最重要）**
   - 外部サービス（Google Maps, メール送信ツール, 予約システムなど）と連携するための「APIキー」や「パスワード」は、**絶対に `index.html` や `main.js` の中に直接書いてはいけません。**
   - HTML等のファイルに書かれた内容は、ブラウザで「ソースを表示」するだけで世界中の誰でも見ることができてしまうためです。
   - どうしても必要な場合は、ホスティングサービス（NetlifyやVercel等）の「環境変数設定」などの安全な裏側の仕組みを使います。
2. **外部スクリプトの安易な読み込み禁止**
   - `<script src="...">` でよくわからない外部サイトのコードを読み込まないようにします。公式で安全性が確認されているもの（Google Fontsなど）のみを使用します。
3. **フォームのスパム対策**
   - お問い合わせフォームなどを自作する際は、そのまま設置するとスパムメールの標的になります。Google Formsや、セキュリティ対策が施された専用のフォームサービス（Netlify Forms, Formspreeなど）を利用します。

### サイト構造（コーポレート・支援共感版構造案）
「個人（当事者）に『自分にもできそう、やりたい』と思わせる共感」と「将来の支援団体（B2B/B2G）に向けた信頼感」を両立させる構成です。

- [ ] **1. Hero**: 「霧が晴れ、自分の『運転感覚』を取り戻す」自然で透明感のあるビジュアルとメッセージ。
- [ ] **2. Pain & Empathy (共感)**: 「能力はあるのに、なぜかノイズに邪魔される」「何かに挑戦したいのに、意欲が続かない」といった、グレーゾーンやひきこもりの方が抱える具体的な「もどかしさ」に寄り添うセクション。
- [ ] **3. Mission / Vision**: なぜSecond Memoryなのか。脳のハンドルを自分の手に取り戻す世界。
- [ ] **4. The Mechanism (なぜ効くのか？)**: （※学術的すぎないレベルで）身体の動き（インプット）が脳を調律し、ノイズを消して「意欲」を生み出すメカニズムを簡潔に説明。
- [ ] **5. Product (Self-Driver)**: 室内で完結する4つのコア・トレーニング（バランス、指の巧緻性、高速音読、逆暗唱）。「1日30分・数ヶ月の継続が、一生のコンディションを作る」という事実を誠実に伝える。
- [ ] **6. For Supporters (支援者・団体の方へ)**: B2B/B2Gを見据え、ひきこもり支援等に活用できる旨を記載するセクション。
- [ ] **7. Company**: 会社概要（Second Memory株式会社について）。
- [ ] **8. Contact / Booking**: 問い合わせ案内。各所に配置（体験へのスムーズな導線）。
